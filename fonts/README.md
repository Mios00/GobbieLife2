# JetBrains Mono (self-hosted)

The `.woff2` files here are subsets of **JetBrains Mono**, licensed under the
**SIL Open Font License 1.1** (<https://openfontlicense.org>). Source:
<https://www.jetbrains.com/lp/mono/> / Google Fonts.

`jetbrains-mono.css` is the Google Fonts `css2` output for weights 400–800
(normal) + 400 (italic), with every `fonts.gstatic.com` `url()` rewritten to
the local files in this directory — so the game loads its type with no external
network request.
